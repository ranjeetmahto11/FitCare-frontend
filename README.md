# FitCare — Frontend

A clean and responsive fitness goal tracking
frontend built with HTML, Bootstrap 5 and
Vanilla JavaScript.

## 🔗 Live Demo

- **Frontend:** https://fitcare-app.netlify.app
- **Backend API:** https://fitcare-backend-production.up.railway.app
- **Swagger UI:** https://fitcare-backend-production.up.railway.app/swagger-ui.html
- **GitHub Backend:** https://github.com/ranjeetmahto11/FitCare-backend

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Custom dark theme styling |
| Bootstrap 5.3 | UI components and layout |
| Bootstrap Icons | Icons throughout app |
| Vanilla JavaScript | Logic and API calls |
| Fetch API | HTTP requests to backend |

## ✨ Features

- 🔐 User Registration and Login with JWT
- 🎯 Fitness Goal Selection (7 goals)
- 🗺️ Personalized Roadmap with Phases
- 📊 Goal Progress Tracking
- ⚖️ BMI Calculator
- 🌑 Dark Theme UI
- 📱 Fully Responsive Design
